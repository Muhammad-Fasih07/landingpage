## Overview

Employee Activity Detector combines a Python monitoring agent with a MERN dashboard to surface idle time and application usage insights.

## Architecture

- **Agent:** Python process for idle and application usage detection
- **API:** Node.js / Express ingestion and query endpoints
- **Dashboard:** React UI with charts and exportable reports
- **Storage:** MongoDB activity logs

## Responsibilities

- Built detection logic and reporting pipelines
- Designed dashboard views for logs and analytics
- Balanced visibility with practical, privacy-aware defaults

## Outcome

A pragmatic monitoring stack that pairs a lightweight agent with a clear engineering dashboard.
