## Overview

Hike-Connect is a cross-platform mobile app for discovering and organizing hiking events - public, private, and featured meetups.

## Architecture

- **Mobile:** React Native for iOS/Android UX
- **API:** Node.js / Express REST services
- **Data:** MongoDB for users, events, and RSVPs
- **State:** Redux on the client for predictable UI state

## Responsibilities

- Built event creation, discovery, and RSVP flows
- Implemented authentication and profile features
- Designed API contracts between mobile and backend

## Outcome

A complete mobile + API product demonstrating end-to-end MERN and React Native delivery.
